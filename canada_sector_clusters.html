Methodology &
Data Sources

An overview of the sovereign capability taxonomy, NAICS code mapping, and the spatial clustering methodology used to identify defence-industrial firm clusters across Canada.

This dashboard maps Canadian firms engaged in sovereign defence-industrial activities to a set of macro capability domains defined by the Government of Canada. Firms are identified using 6-digit NAICS codes associated with each sub-capability, then spatially clustered to reveal geographic concentrations of industrial capacity.

The analysis draws on geocoded business register data, projected to the Statistics Canada Lambert Conformal Conic projection (EPSG:3347) for accurate distance-based clustering, then reprojected to WGS84 for cartographic display.

DBSCAN Spatial Clustering

Density-Based Spatial Clustering of Applications with Noise

DBSCAN groups points that are closely packed together, marking isolated points as noise. Unlike k-means, it requires no pre-specified number of clusters and can find clusters of arbitrary shape, which is well suited to Canada's irregular industrial geography.

A firm is a core point if at least minPts firms fall within distance ε. Firms reachable from a core point join its cluster, while all others are noise and excluded from the visualisation.

ParameterValueRationale
ε (epsilon)10,000 mCaptures firms within a commutable industrial zone
min_samples10 firmsSuppresses isolated single-firm locations
ProjectionEPSG:3347Statistics Canada Lambert — preserves distances
Noise label−1Excluded from map polygons

Convex Hull Visualisation

Geographic footprint of each cluster

Each DBSCAN cluster is shown as a convex hull — the smallest convex polygon containing all firm locations in that cluster. This preserves geographic extent without exposing individual firm locations.

Hulls are computed using Shapely's convex_hull on the union of firm geometries per cluster, then reprojected to WGS84 for rendering. Clusters with fewer than three firms are omitted.
